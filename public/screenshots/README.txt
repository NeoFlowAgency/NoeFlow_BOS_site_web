COMMENT FAIRE DE BONS SCREENSHOTS
===================================
1. Mets ton navigateur en plein écran (F11)
2. Zoom navigateur à 100% (Ctrl+0)
3. Utilise des données réalistes (pas "test" ou "aaa")
4. Sur Mac : Cmd+Shift+4 pour sélectionner la zone
   Sur Windows : Win+Shift+S ou outil Snipping Tool
5. Exporte en PNG, renomme selon les noms ci-dessus
